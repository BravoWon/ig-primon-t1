The Kernel GATE_KERNEL.md

One dimension-free schema, instantiated fifteen times. Every gate fills the same slots: a falsifiable claim, a chaos reference or surrogate null, in-run anchors on exactly known values, rotation controls, verdicts committed before the first number exists, and a non-measurement discipline — an instrument that cannot see is never allowed to testify.

LAW 1
Instrument bug until proven otherwise. Every anomaly is prosecuted as instrument error first. Picard saturation, axis slivers, float-cast contamination, plateau-poisoned fitters — all caught this way.
LAW 2
Universality is anonymity. If a statistic is universal (GUE spacings, scaling exponents), it cannot identify its source. Claims that decode the universal part are refuted by construction.
LAW 6
Auto-verdicts are advisory — read the table. A fitter is an instrument too. Its number is disowned the moment inspection shows the rule it implements is not the rule that was registered.
LAW 8
The failure is a measurement. A wall, mapped in triplicate with receipts, is banked with the same ceremony as an anchor hit. Meaning is the residue off the fixed point.
